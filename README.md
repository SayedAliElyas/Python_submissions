# Python Practice — NeetCode

Solutions to coding problems from [NeetCode.io](https://neetcode.io), practiced as preparation for internship and working student applications. Focus is on data structures, algorithms, and writing clean, working Python — not on producing finished projects.

## Structure

Solutions are organized by topic, then by problem:

```
<topic>/<problem-id>/submission-0.py
```

Each `submission-N.py` is a new attempt at the same problem — later submissions are revised or improved versions of earlier ones, not separate solutions.

Each file contains a function solving the problem (matching NeetCode's submission format), not a standalone script.
